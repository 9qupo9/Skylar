# Skylar

Этот проект представляет собой игровой интерфейс на React для системы сундуков Skylar. Он включает в себя пользовательский интерфейс для взаимодействия с игрой в сундуки, управления балансом пользователя, подключения кошелька и управления инвентарем наград. Приложение содержит компоненты для отображения сундуков, системы наград и интеграции кошелька.
